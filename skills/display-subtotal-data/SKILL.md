# Power BI Visual — Display Subtotal Data

The Total/Subtotal API enables Power BI matrix visuals to display row and column totals and subtotals. This feature is only applicable to matrix-type visuals and is configured through the `subtotals` property in `capabilities.json`.

## Trigger

**Use when:**

- The user asks to implement power bi visual — display subtotal data in a Power BI custom visual.
- The user needs guidance on the display-subtotal-data feature or API.

**Don't use when:**

- The user is working with a standard (non-custom) Power BI visual that does not support the visuals SDK.
