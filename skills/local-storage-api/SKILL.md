# Power BI Visual — Local Storage API

The Local Storage API allows a Power BI custom visual to store data in the browser's local storage. This enables persisting state (like user preferences, counters, or cached data) between visual sessions. Each visual type has isolated storage access.

## Trigger

**Use when:**

- The user asks to implement power bi visual — local storage api in a Power BI custom visual.
- The user needs guidance on the local-storage-api feature or API.

**Don't use when:**

- The user is working with a standard (non-custom) Power BI visual that does not support the visuals SDK.
