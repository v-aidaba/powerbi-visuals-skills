# Power BI Visual — Landing Page

A landing page is displayed in a Power BI custom visual when no data has been mapped to it. It can show usage instructions, links to documentation, videos, or any helpful content to guide the user. The landing page disappears once data is added.

## Trigger

**Use when:**

- The user asks to implement power bi visual — landing page in a Power BI custom visual.
- The user needs guidance on the landing-page feature or API.

**Don't use when:**

- The user is working with a standard (non-custom) Power BI visual that does not support the visuals SDK.
