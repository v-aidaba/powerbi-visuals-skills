# Power BI Visual — Add Localization

The Localization API enables a Power BI custom visual to display text in the user's language. By providing string resource files for different locales, the visual automatically adapts to the language configured in Power BI.

## Trigger

**Use when:**

- The user asks to implement power bi visual — add localization in a Power BI custom visual.
- The user needs guidance on the add-localization feature or API.

**Don't use when:**

- The user is working with a standard (non-custom) Power BI visual that does not support the visuals SDK.

## Overview

The Localization API enables a Power BI custom visual to display text in the user's language. By providing string resource files for different locales, the visual automatically adapts to the language configured in Power BI.
