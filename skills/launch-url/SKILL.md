# Power BI Visual — Launch URL

The Launch URL API allows a Power BI custom visual to open a URL in a new browser tab using `host.launchUrl()`. This is useful for linking to documentation, external resources, or data source details.

## Trigger

**Use when:**

- The user asks to implement power bi visual — launch url in a Power BI custom visual.
- The user needs guidance on the launch-url feature or API.

**Don't use when:**

- The user is working with a standard (non-custom) Power BI visual that does not support the visuals SDK.
