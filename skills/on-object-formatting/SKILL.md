# Power BI Visual — On-Object Formatting & Subselection API (Preview)

On-object formatting allows users to modify the format of visual elements by directly selecting them on the canvas. When an element is selected, Power BI shows a context menu/toolbar and the format pane automatically navigates to the relevant setting.

## Trigger

**Use when:**

- The user asks to implement power bi visual — on-object formatting & subselection api (preview) in a Power BI custom visual.
- The user needs guidance on the on-object-formatting feature or API.

**Don't use when:**

- The user is working with a standard (non-custom) Power BI visual that does not support the visuals SDK.
